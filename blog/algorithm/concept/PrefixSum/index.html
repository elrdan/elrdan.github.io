누적합(Prefix Sum, Cumulative Sum)​

누적합은 주어진 배열이나 리스트의 각 요소까지 합을 구하여 새로운 배열이나 리스트를 생성하는 방법입니다.
이 방식은 데이터의 누적 변화를 분석하거나, 특정 시점까지의 합을 빠르게 계산할 때 유용합니다.

누적합의 공식​

주어진 배열[a1,a2,...,ana_1, a_2,...,a_na1​,a2​,...,an​]이 있을때, 누적합 배열[s1,s2,...,sns_1, s_2,..., s_ns1​,s2​,...,sn​]은 다음과 같이 정의됩니다.

si=a1+a2+...+ais_i = a_1 + a_2 + ... + a_isi​=a1​+a2​+...+ai​

즉, 누적합 배열의 각 요소는 원래 배열의 처음부터 해당요소까지의 합을 나타냅니다.

누적합 활용 예시​

  1. 시간에 따른 데이터 분석: 매출 데이터, 기온 변화 등 시간에 따른 누적 변화를 분석할 때 유용합니다.
  2. 구간 합 계산: 특정 구간의 합을 빠르게 계산할 수 있습니다. 예를 들어, 누적합 배열을 사용하면[i,ji, ji,j] 구간의 합을 sj−si−1s_j - s_{i-1}sj​−si−1​로 계산할 수 있습니다.
  3. 데이터 시각화: 누적 그래프를 통해 데이터의 변화를 직관적으로 파악할 수 있습니다.

예시 문제​

다음 배열이 주어졌을 때, 누적합 배열을 구하시오.

입력 1, 2, 3, 4, 5

출력 1, 3, 6, 10, 15

코드 구현​

예시 문제 풀이

public class PrefixSum {
	public static void main(String[] args) {
		int i = 2;
		int j = 5;
		int[] arr = {1, 3, 5, 7, 9 , 11};
		int[] arrSum = new int[arr.length];

		prefixSum(arr, arrSum);

		System.out.println(arrSum[j] - arrSum[i-1]);
	}

	static void prefixSum(int[] arr, int[] arrSum) {
		arrSum[0] = arr[0];
		for(int i=1; i<arr.length; i++) {
			arrSum[i] = arrSum[i-1] + arr[i];
		}
	}
}
