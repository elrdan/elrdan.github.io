## 진수
  1. 진수는 특정 개수의 숫자만을 이용하여 수를 나타내는 수 체계입니다.
## 진수 변환
  1. 10진수를 n진수로 변환
     - 10진수 값의 몫이 n보다 작을 때까지 n으로 나누고 나머지 값들을 표시한 후 나머지 갓들을 읽습니다.     
      <br/>
      ``` 
      10진수 18을 2진수로 변환
        
      18 / 2 -> 0
      9 / 2 -> 1
      4 / 2 -> 0
      2 / 2 -> 0
      1
    
      10진수의 몫이 n보다 작은 1이 되었기때문에 나머지 값들을 역순으로 읽어주면 됩니다.
        
      [결과] 10010
      ```
  2. **n진수를 10진수로 변환**
     - n진수의 값을 마지막 자릿수 $$\times$$ $$n^0$$ 오른쪽에서 2번째 자릿수 $$\times$$ $$n^1$$ 와같이 모든 자릿수를 처리하고 곱해준 값을 모두 더해 줍니다.
     <br/>
       ```
       2진수 18(10010)을 10진수로 변환
    
       1 * 2^1 = 2
       1 * 2^4 = 16
    
       16 + 2
    
       [결과] 18 
       ```
       <br/>
## 아스키 코드
  1. **아스키 코드(ASCII; American Standard Code Information Interchange)**
     - 아스키 코드는 미국 ANSI에서 표준화한 정보교환용 부호체계입니다.
     - 영문 키보드로 입력할 수 있는 모든 기호가 할당된 기본적인 부호체계입니다.
  2. **주요 아스키 코드**
      |10진수|부호|10진수|부호|10진수|부호|

      |0|NULL|65|A|97|a|
      |32|''(Space)|48|0|
